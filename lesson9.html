Project Name: Engineering Final Project
Date: October 26, 2025

Summary

This project focuses on designing and fabricating a motor mount for a robotic arm. The main challenge was creating a lightweight yet durable mount that could withstand the motor's torque and vibrations. We explored several designs, from simple 3D-printed brackets to more complex machined parts.

Figure 2: Close-up of the motor mount assembly.

Iteration

After testing the initial prototype, we identified several areas for improvement. The original design showed signs of stress on the mounting points, so we reinforced the structure and added a second support bracket to distribute the load more evenly.

Figure 3: CAD model of the revised, reinforced motor mount.

Conclusion

The final design successfully met all project requirements. The motor mount is stable, lightweight, and easy to assemble. The use of external CSS and a clear page layout made the final report easy to read and understand, fulfilling the project's documentation goals.
