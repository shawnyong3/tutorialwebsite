Lesson 8: What CSS Does

HTML provides the structure of a webpage (what content is there). CSS provides the style (how it looks).

Example: Plain HTML (no CSS)

Hello World

This is a paragraph of plain text. Nothing fancy here.

Example: HTML with CSS

Hello World

This is the same paragraph, but styled with CSS for color, spacing, and emphasis.

Notice how CSS changes fonts, colors, spacing, alignment, and emphasis.
